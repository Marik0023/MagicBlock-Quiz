# MagicBlock Quizzes (GitHub Pages)

This is a simple static website template for GitHub Pages.

## Structure
- `index.html` — landing page with 3 quiz cards
- `styles.css` — styling + animated pixel overlay
- `quizzes/` — quiz pages + JS
- `assets/` — background, audio clips, movie frames

## Deploy (GitHub Pages)
1. Create a new GitHub repo
2. Upload all files from this folder
3. GitHub repo → Settings → Pages
4. Source: Deploy from a branch → Branch: `main` → Folder: `/root`
5. Open the provided Pages URL

## Add your content
- Audio clips: `assets/audio/clip1.mp3`, etc
- Frames: `assets/frames/frame1.jpg`, etc
- Edit quiz data in:
  - `quizzes/song.js`
  - `quizzes/movie.js`
  - `quizzes/magicblock.js`
